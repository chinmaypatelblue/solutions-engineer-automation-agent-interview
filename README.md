# AI-Assisted Client Operations Take-Home

Start here.

## Candidate Deliverable

Upload a private 5-8 minute demo video to YouTube, Dailymotion, or the video hosting service of your choice, and send us the link.

No source code, repository, README, deployment URL, or written document is required from the candidate. The video should show the product working and briefly explain the approach.

## Files in this package

- `ASSIGNMENT.md` - full candidate-facing assignment instructions.
- `DEMO_RECORDING_CHECKLIST.md` - checklist for what the demo recording should cover.
- `PROTOTYPE_NOTES.md` - notes about what is and is not included in the static prototype.
- `prototype/client-operations-portal.html` - sanitized static HTML prototype to use as the starting point.

## High-level task

Turn the static Client Operations Portal prototype into a functional app and demonstrate an AI-assisted workflow where Sales can request changes through a chat or agent interface, preview them in UAT, approve them, promote them to production, and optionally roll them back.

## Important note

All data in the prototype is synthetic and sanitized. Client names, IDs, dates, values, vendors, and records are fictitious.
The provided materials have no connection to real assets, real instruments, real clients, or real prices.
Any similarity or comparison to actual assets, securities, clients, symbols, records, or market prices is purely coincidental.
