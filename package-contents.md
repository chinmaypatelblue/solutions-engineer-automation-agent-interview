# Package Contents

This candidate package contains:

1. `README.md`
   - Quick start and package overview.

2. `assignment.md`
   - Candidate-facing assignment instructions.
   - Deliverable requirement: private 5-8 minute demo video link only.
   - Scope, timebox, build requirements, and evaluation criteria.

3. `demo-checklist.md`
   - Checklist candidates can use while preparing the demo video.

4. `prototype-notes.md`
   - Notes about the static prototype, external CDN dependencies, and simulated systems.

5. `prototype/client-operations-portal.html`
   - Sanitized static HTML prototype.
   - Includes Client Database as the main page.
   - Includes Agreement Intake as the secondary page.
   - Includes header, sidebar, breadcrumbs, styling, synthetic data, and simple page switching.

This package does not include an internal evaluation rubric, so it is safe to send directly to candidates.
