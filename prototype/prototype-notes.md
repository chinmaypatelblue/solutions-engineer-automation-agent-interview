# Prototype Notes

The provided HTML file is intentionally static. It is meant to be a starting point, not a finished application.

## What is included

- A presentable admin-style shell with sidebar, header, and breadcrumbs.
- Client Database page.
- Agreement Intake page.
- Sanitized, fictitious client records.
- A simple JavaScript page switcher.
- Bootstrap 5, Bootstrap Icons, and Google Fonts loaded from public CDNs.

## What is not included

- Backend API.
- Database.
- Authentication.
- Real file upload.
- Real AI extraction.
- Real chat integration.
- Real UAT or production deployment.

You may implement or simulate any of these pieces. The demo should make it clear what is real and what is simulated.

## Important

All client names, IDs, dates, and financial values in the prototype are fictitious and sanitized.
